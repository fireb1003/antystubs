## Install apollo

`npm install react-router-dom apollo-boost @apollo/react-hooks graphql apollo-cache-inmemory apollo-client apollo-link-http apollo-fetch moment`

## Adding ant framework

`npm install antd`

## Add Routing

`npm install @types/react-router-dom`
